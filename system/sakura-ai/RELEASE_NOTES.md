# SAKURA AI Chat - Release Notes

## [v0.52] - 2026-03-11
### 修正・改善
- **最新ニュース回答拒否の完全修正**: システムが外部（ニュース等）から情報を取得したにも関わらず、AI側の「私にはリアルタイムの情報を取得する能力がありません」という安全フィルターが発動して回答を拒否する問題を完全に修正しました。今後は取得した情報をシステム上の『絶対の事実』として認識し、ユーザーに対して的確に要約して回答するようになります。
- **ニュース取得の並列化（高速化）**: 複数のニュースソースを一括で取得する際、順番に処理することによるタイムアウト（時間切れ）を防ぐため、裏側で情報を同時に取得する（並列処理）仕組みを導入しました。これにより多数のニュースサイトとも安定して連携可能になりました。

## [v0.51] - 2026-03-11
- **𝕏(Twitter) 連携機能の廃止**: ノイズ混入や検索制限の課題があった𝕏連携をUI・システム両面から完全に削除しました。
- **ニュースソースの大幅拡張**: トップニュースの取得元を「Yahoo!ニュース」単体から、**livedoor ニュース、NHKニュースWeb、読売新聞、毎日新聞を含めた5媒体**に大幅拡張しました。これにより、AIがより多角的で幅広い最新報道・トレンドを把握した上で回答できるようになりました。

## [v0.50] - 2026-03-11
### 修正・改善
- **AIの「情報提供拒否」の解消**: AIが「私はAIなのでリアルタイム情報は提供できません」と頑なに回答を拒否する問題（モデルの安全フィルターの過剰反応）を根絶するため、システム指示を「オーバーライド司令」として再定義し、取得した外部データ（ニュースやトレンド）を絶対的な事実として解説するように強制しました。
- **Yahooニュースの対応ジャンル拡大**: Yahooニュースの取得対象を「主要ニュース」だけでなく「IT・トレンド」「エンタメ」等の複数ジャンルに拡大しました。
- **𝕏 トレンドと個人検索の並列化**: 𝕏検索において、ご自身の最新投稿と、世間の一般的なトレンド検索を同時に行えるように内部処理を大幅に進化させました。

## [v0.48] - 2026-03-11
### 修正・改善
- **𝕏 検索条件の緩和**: 前回のアップデートで検索条件を厳格にしすぎた結果、𝕏上の投稿自体が検索システムから全く取得できなくなり「最新情報が取得できませんでした」と返事してしまう問題を修正しました。本人確認の精度を保ちつつ、より確実に最新の投稿を拾えるように検索クエリを調整しました。

## [v0.47] - 2026-03-11
### 修正・改善
- **𝕏 制限・精度の大幅向上**: 𝕏（Twitter）検索を行う際、同名の別アカウント（Instagramのシックスパック写真やYouTubeの韓国旅行等）の情報が混入してしまう問題を解決しました。AIに対して「InstagramやYouTubeの情報は別人として完全に無視する」強力なプロンプトを追加し、Web検索自体からも他SNSの情報を物理的に除外しました。

## [v0.46] - 2026-03-11
### 修正・改善
- **ニュース連携の強化（詳細情報の取得）**: Yahooニュースの記事タイトルだけでなく、記事の要約（概要）もAIに渡すように改良しました。これにより、より深くニュースの内容を踏まえた回答が可能になりました。
- **情報ソース表示の不具合修正**: ニュース、𝕏 ポスト、検索結果が同時に存在する場合に、一部が表示されない問題を修正しました。すべての情報源が正しく画面に表示されます。
- **AIの回答モード最適化**: AIが最新情報を拒否せず、提供されたデータを確実に回答に反映させるための指示を強化しました。

## [v0.45] - 2026-03-11
### 修正・改善
- **リアルタイム情報取得の安定化（cURL化）**: Yahooニュース、天気、𝕏 ポストなどの外部データ取得方式を改善し、より確実に最新情報を取得できるようにしました。
- **AIの回答品質向上**: AIが最新情報を無視したり「能力がない」と答えたりする問題を改善し、提供されたニュースや検索結果を積極的に回答へ反映するように指示を強化しました。

## [v0.37] - 2026-03-11
### 新機能
- **𝕏 パーソナライズ設定機能の追加**: 設定（Settings）画面に 𝕏 のユーザー名と関心事を入力できる項目を追加しました。これにより、𝕏 連携ボタンをオンにした際の検索精度が大幅に向上し、ユーザー自身の投稿や関連性の高い情報を優先的に拾えるようになりました。

## [v0.36] - 2026-03-11
### 新機能
- **「𝕏 ポスト」ボタンの追加 (X/Twitter 連携)**: ヘッダーメニューに「𝕏 ポスト」ボタンを追加しました。このボタンをオンにすると、AI が X (旧Twitter) 上の最新ポストや口コミを検索し、リアルタイムな世間の反応を回答に反映できるようになりました。Tavily API のドメイン指定検索を活用しており、高額な API 契約なしで SNS の動向をキャッチできます。

## [v0.35] - 2026-03-11
### 新機能
- **「📰 ニュース」ボタンの追加（Yahooニュース連携）**: 画面右上のヘッダーメニューに「📰 ニュース」ボタンを追加しました。このボタンをオンにして質問すると、AIが最新のYahooトップニュースの見出しをリアルタイムで取得し、それを踏まえた上でユーザーの質問（「今日のニュースは？」「最近の話題について」など）に回答できるようになりました。取得したニュースの参照元URLは、回答内に折りたたみUIとして表示されます。

## [v0.34] - 2026-03-11
### 修正・改善
- **インライン参照リンクの折りたたみUI化**: AIが自主的に出力した「【参考ウェブサイト】」のリスト群を完全に削除するのではなく、会話文の途中にあっても自動で「折りたたみ可能なUI」に変換してコンパクトに表示する機能を追加しました。本文の視認性を保ちつつ、必要に応じてソースを確認できるようになりました。

## [v0.33] - 2026-03-11
### 修正・改善
- **AI側出力の強制的ミュート（フロントエンド対策）**: AIが意図せず「【参考ウェブサイト】」といった余計なテキストを自分で生成してしまった場合でも、表示用のプログラム（JavaScript）側でその部分を自動的に検知・切除し、画面には一切表示されないようにする安全装置を導入しました。

## [v0.32] - 2026-03-11
### 修正・改善
- **検索結果重複表示の完全修正**: 検索ソースのURLをシステム裏側で処理し、AIに渡さないことで自主的なURL出力を確実に防止しました。
- **キャッシュ対策の強化**: ブラウザに古いJavaScriptがキャッシュされ続ける問題を防ぐため、強制キャッシュクリアの`meta`タグを追加しました。
