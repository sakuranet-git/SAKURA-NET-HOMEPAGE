# Gemini 依頼: さくらねっとHP ロゴ・ファビコン・アプリアイコン作成

## 依頼概要

コーポレートサイト `https://sakuranet-co.jp/` で使用する以下3点の画像を作成してください。
SEOの構造化データ (schema.org Organization) および PWA / モバイルブックマークで利用します。

---

## 📋 作成対象（3点）

### ① `logo.png` — schema.org Organization logo 用

| 項目 | 指定 |
|---|---|
| ファイル名 | `logo.png` |
| サイズ | **512×512px（正方形）** |
| フォーマット | PNG（背景透過） |
| 用途 | Google検索結果のOrganizationリッチカードに表示されるロゴ |
| 要件 | Google推奨：最低112×112px以上、正方形または近い比率。ロゴ全体が見切れずに収まること |

### ② `apple-touch-icon.png` — iOS ホーム画面追加時のアイコン

| 項目 | 指定 |
|---|---|
| ファイル名 | `apple-touch-icon.png` |
| サイズ | **180×180px（正方形）** |
| フォーマット | PNG（背景塗りつぶし推奨・`#c82054` 桜ピンク or `#fff0f5`） |
| 用途 | iPhone / iPad でホーム画面に追加したときのアイコン |
| 要件 | 角丸はiOS側で自動処理される。背景透過ではなく塗りつぶし推奨 |

### ③ `favicon.ico` — ブラウザタブ用ファビコン

| 項目 | 指定 |
|---|---|
| ファイル名 | `favicon.ico` |
| サイズ | **マルチサイズ (16×16, 32×32, 48×48 を1ファイルに格納)** |
| フォーマット | ICO（マルチレイヤー） |
| 用途 | ブラウザのタブ・ブックマーク・履歴で表示されるアイコン |
| 要件 | 小さくても識別できるシンプルな図案にすること（文字は読めなくてよい・シンボル優先） |
