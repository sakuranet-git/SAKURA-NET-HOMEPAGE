# Gemini 依頼: さくらねっとHP OGP画像デザイン

## 依頼概要

コーポレートサイト `https://sakuranet-co.jp/` の SNS共有用OGP画像（og:image）を作成してください。

## 仕様

| 項目 | 内容 |
|---|---|
| サイズ | **1200 × 630 px**（OGP標準） |
| 書き出し | JPG（品質85以上・200KB以下が理想） |
| ファイル名 | `ogp.jpg` |
| 配置先 | `/img/ogp.jpg`（最終的に `https://sakuranet-co.jp/img/ogp.jpg`） |
| 用途 | Facebook / X（Twitter） / LINE / Slack などSNSシェア時のサムネイル |

## デザイン方針

### ブランドカラー（必ず使用）
- メイン: `#c82054`（深いピンク）
- アクセント: `#e03164`（明るいピンク）
- ベース: `#fff0f5` or `#ffffff`（薄いピンク白 / 白）

### レイアウト要件
- **左側**: 会社名・キャッチコピー
  - 会社名: 「株式会社さくらねっと」または「SAKURA-NET Inc.」
  - キャッチコピー: **「UniFi × SAKURA-NET光 で、次世代の企業ネットワークを。」**
  - サブコピー（任意）: 「大阪・全国対応 / 通信・セキュリティ・空間管理を1画面で」
- **右側 または 背景**: UniFiネットワーク機器 or モダンなオフィスの抽象ビジュアル
- **ロゴ**: 会社ロゴがあれば左上 or 右下に控えめに配置
- **セーフエリア**: 中央の 1000×500px 以内に主要情報を収める（LINEサムネ対応）

### テイスト
- 洗練されたビジネス向け
- 信頼感・先進感・清潔感
- 花やピンクグラデーションで柔らかさも演出（ただし女性的すぎない）

### NG例
- 文字が多すぎる（スマホサムネで読めない）
- 明度の低い写真ベース（テキストが埋もれる）
- 過度な装飾・派手すぎる色
- 著作権クリアでないストック素材の流用

## 出力形式

1. **OGP画像本体** (`ogp.jpg` / 1200x630px)
2. **プレビュー** (SNSでどう見えるかのモックアップ画像・任意)
3. **デザイン意図の説明**（50字以内）
