# Gemini 差し戻し依頼: favicon.ico 再作成

## 依頼概要

先ほど納品いただいた `favicon.ico` が **16×16 の単一サイズのみ** となっており、当初の仕様（マルチサイズ）と差異があるため再作成をお願いします。

`logo.png` (512×512) と `apple-touch-icon.png` (180×180) は仕様どおり問題なく受領しましたので、**favicon.ico のみ** の差し替えとなります。

---

## 🔴 問題点

| 項目 | 報告内容 | 実ファイル |
|---|---|---|
| サイズ | 16×16, 32×32, 48×48 マルチサイズ | **16×16 のみ** |
| ファイルサイズ | 約 0.6KB | 657バイト（1レイヤーなので妥当） |

ブラウザタブ・高DPIディスプレイ・ブックマーク・Windowsタスクバーなどで**ボケ・劣化表示**になる可能性があります。

---

## 📋 再作成仕様（favicon.icoのみ）

| 項目 | 指定 |
|---|---|
| ファイル名 | `favicon.ico` |
| 必須サイズ | **16×16 + 32×32 + 48×48 の3レイヤーを1ファイルに格納** |
| フォーマット | ICO（マルチレイヤー・BMP/PNG圧縮どちらでも可） |
| 用途 | ブラウザタブ・ブックマーク・履歴・OSアイコン表示 |
| デザイン | 既存の `logo.png` と同一デザイン（縮小版） |
| 要件 | 16×16でも識別できるよう、小さいサイズ用に**細部を簡略化**してもよい（シンボル優先・文字は読めなくてよい） |

---

## 💡 技術的補足（推奨手法）

Python (Pillow) を使う場合の参考コード：
```python
from PIL import Image
img = Image.open('logo.png')  # 512x512 RGBA
img.save('favicon.ico', format='ICO',
         sizes=[(16, 16), (32, 32), (48, 48)])
```

ImageMagick を使う場合：
```bash
magick logo.png -define icon:auto-resize=48,32,16 favicon.ico
```

※ 16×16 サイズは自動縮小だとディテールが潰れやすいため、**16×16専用に手動で調整**いただけると理想的です。
